African-American Periodicals: a Visual Bibliography

Select a decade and topic from the drop-down menus to see where relevant periodicals were published. City circles are red when we have a link to at least one periodical archive from that location. Click cities to learn more; click states to zoom in.

All initial data comes from African-American newspapers and periodicals: a national bibliography. I have three (unattainable) goals for this project: to fix all data-mining mistakes; to add the 21st century; and to link to an archive of every periodical in the dataset. If you would like to contribute to any of these goals, please do! The project is on github and the dataset is here. If you are not comfortable with git/github, but would still like to report an issue, you can contact me through e-mail.
